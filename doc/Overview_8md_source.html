1 **ARPACK-Armadillo** is a redesign of the [ARPACK](http://www.caam.rice.edu/software/ARPACK/)
2 software for large scale eigenvalue problems, built on top of
3 [Armadillo](http://arma.sourceforge.net/), an open source C++ linear algebra library.
4 
5 **ARPACK-Armadillo** is implemented as a header-only C++ library which relies on
6 the BLAS and LAPACK libraries. Therefore program that uses **ARPACK-Armadillo**
7 should also link to those two libraries.
8 
9 ## Common Usage
10 
11 **ARPACK-Armadillo** is designed to calculate a specified number (\f$k\f$) of eigenvalues
12 of a large square matrix (\f$A\f$). Usually \f$k\f$ is much less than the size of matrix
13 (\f$n\f$), so that only a few eigenvalues and eigenvectors are computed, which
14 in general is more efficient than calculating the whole spectral decomposition.
15 
16 To use the eigen solvers in this library, the user does not need to directly
17 provide the whole matrix, but instead, the algorithm only requires certain operations
18 defined on \f$A\f$, and in the basic setting, it is simply the matrix-vector
19 multiplication. Therefore, if the matrix-vector product \f$Ax\f$ can be computed
20 efficiently, which is the case when \f$A\f$ is sparse, **ARPACK-Armadillo**
21 will be very powerful for large scale eigenvalue problems.
22 
23 There are two major steps to use the **ARPACK-Armadillo** library:
24 
25 1. Define a class that implements a certain matrix operation, for example the
26 matrix-vector multiplication \f$y=Ax\f$ or the shift-solve operation
27 \f$y=(A-\sigma I)^{-1}x\f$. **ARPACK-Armadillo** has defined a number of
28 helper classes to quickly create such operations from a matrix object.
29 See the documentation of DenseGenMatProd, DenseSymShiftSolve, etc.
30 2. Create an object of one of the eigen solver classes, for example `SymEigsSolver`
31 for symmetric matrices, and `GenEigsSolver` for general matrices. Member functions
32 of this object can then be called to conduct the computation and retrieve the
33 eigenvalues and/or eigenvectors.
34 
35 Below is a list of the available eigen solvers in **ARPACK-Armadillo**:
36 - SymEigsSolver: for real symmetric matrices
37 - GenEigsSolver: for general real matrices
38 - SymEigsShiftSolver: for real symmetric matrices using the shift-and-invert mode
39 - GenEigsRealShiftSolver: for general real matrices using the shift-and-invert mode,
40 with a real-valued shift
41 
42 ## Examples
43 
44 Below is an example that demonstrates the use of the eigen solver for symmetric
45 matrices.
46 
47 ~~~~~~~~~~{.cpp}
48 #include <armadillo>
49 #include <SymEigsSolver.h> // Also includes <MatOp/DenseGenMatProd.h>
50 
51 int main()
52 {
53  // We are going to calculate the eigenvalues of M
54  arma::mat A = arma::randu(10, 10);
55  arma::mat M = A + A.t();
56 
57  // Construct matrix operation object using the wrapper class DenseGenMatProd
58  DenseGenMatProd<double> op(M);
59 
60  // Construct eigen solver object, requesting the largest three eigenvalues
61  SymEigsSolver< double, LARGEST_ALGE, DenseGenMatProd<double> > eigs(&op, 3, 6);
62 
63  // Initialize and compute
64  eigs.init();
65  int nconv = eigs.compute();
66 
67  // Retrieve results
68  arma::vec evalues;
69  if(nconv > 0)
70  evalues = eigs.eigenvalues();
71 
72  evalues.print("Eigenvalues found:");
73 
74  return 0;
75 }
76 ~~~~~~~~~~
77 
78 And here is an example for user-supplied matrix operation class.
79 
80 ~~~~~~~~~~{.cpp}
81 #include <armadillo>
82 #include <SymEigsSolver.h>
83 
84 // M = diag(1, 2, ..., 10)
85 class MyDiagonalTen
86 {
87 public:
88  int rows() { return 10; }
89  int cols() { return 10; }
90  // y_out = M * x_in
91  void perform_op(double *x_in, double *y_out)
92  {
93  for(int i = 0; i < rows(); i++)
94  {
95  y_out[i] = x_in[i] * (i + 1);
96  }
97  }
98 };
99 
100 int main()
101 {
102  MyDiagonalTen op;
103  SymEigsSolver<double, LARGEST_ALGE, MyDiagonalTen> eigs(&op, 3, 6);
104  eigs.init();
105  eigs.compute();
106  arma::vec evalues = eigs.eigenvalues();
107  evalues.print("Eigenvalues found:");
108 
109  return 0;
110 }
